# grue

A Simple URL Shortener

### Tech

- **Express JS**
- **Mongo DB**
- **Vercel** - for the production deployment

### Development

1. Clone the project

```
git clone https://github.com/TheBoringDude/grue.git
```

2. Running it locally (install the Vercel CLI | `npm i -g vercel`)

```
vercel dev
```
